# Void Recall Totem

This datapack modifies the behavior of the **Totem of Undying**.

When a player falls into the void while holding a totem (**in any dimension**), the totem is consumed and the player is saved from death. The player is then teleported to the **Overworld** at a random location within a radius of 3500 blocks from coordinates (0, 0).

## Details

- Works in all dimensions
- Activates only when falling into the void
- Consumes one Totem of Undying on activation
- Teleports the player to a random position in the Overworld (within ±3500 blocks of 0, 0)

This datapack does not modify any other mechanics of the Totem of Undying.
